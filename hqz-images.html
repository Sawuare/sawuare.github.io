HQZ Images

HQZ images are mathematical images which I discovered around 2019. Some of them are in the gallery below. A description of them is in the documentation.

Software

My program hqz-fb displays HQZ images and makes it easy to explore them. My other program hqz-png writes the images, including the ones in the gallery. These programs and more are free and available in my repository.

Favicon

This site's favicon is an HQZ image given by painter 22 and q-argument 15.

Painter 22 reacts to some q-sequences, and q-argument 15 begins one of them: 15, 62, 252, 1016, 4080, 16352… Related sequences include A171472, A171499, A171476, and A020522. My program hqz-seq generates these sequences and more.

Demo

This is an interactive demo of HQZ images. From the options below you can select a painter and input its q-argument to paint a canvas. Note that the options are limited in comparison with those of the main program hqz-png.
