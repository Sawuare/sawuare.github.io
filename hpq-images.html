HPQ Images

HPQ images are mathematical images. I discovered them around 2019. Definition.

Software

  • hpq-fb - display HPQ images on the Linux framebuffer device
  • hpq-png - write an HPQ image to a PNG file
  • hpq-seq - write some HPQ q-sequences
  • hpq.h - code for HPQ images

These programs and libraries are part of misk. I use hpq-fb to explore the images and hpq-png to save them.

Favicon

This site's favicon is an HPQ image given by painter 22 and q-argument 15.

For painter 22, q-argument 15 begins a q-sequence: 15, 62, 252, 1016, 4080, 16352… To see why these values are a sequence, input them in the demo below. Related sequences include A171472 and A171499. hpq-seq generates these sequences and more.
