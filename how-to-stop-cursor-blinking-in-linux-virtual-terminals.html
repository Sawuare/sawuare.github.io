How to Stop Cursor Blinking in Linux Virtual Terminals

In Linux virtual terminals, the cursor may blink by default. This may be distracting and annoying.

On Linux systems, the file /sys/class/graphics/fbcon/cursor_blink contains a number. Wether the number is 1 or 0 determines whether the cursor blinks or not, respectively. The file requires root's permission to be written, and its contents are reset at reboot. Setting the contents to 0 at reboot should stop the blinking. Here I will explain a method for doing this.
