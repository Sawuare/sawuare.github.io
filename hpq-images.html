HPQ Images

HPQ images are mathematical images which I discovered around 2019. Some of them are in the gallery below. A description of them is in the documentation.

Software

My program hpq-fb displays HPQ images and makes it easy to explore them. My other program hpq-png writes the images, including the ones in the gallery. These programs and more are free and available in misc.

Favicon

This site's favicon is an HPQ image given by painter 22 and q-argument 15.

Painter 22 reacts to some q-sequences, and q-argument 15 begins one of them: 15, 62, 252, 1016, 4080, 16352… You can try these values in the demo below. Related sequences include A171472 and A171499. My program hpq-seq generates these sequences and more.

Demo

This is an interactive demo of HPQ images. From the options below you can select a painter and input its q-argument to paint a canvas. Note that the options are limited in comparison with those of the main program hpq-png.
