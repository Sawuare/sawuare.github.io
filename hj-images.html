HJ Images

Description

HJ images are visual patterns that I discovered. A detailed description of them is in this documentation. Some of them are in the gallery below.

Software

The software related to the images is free and available in this repository. The main programs are hj2fb (for viewing images) and hj2png (for saving images).

Check out hj2web for an interactive demo of the images.

Discord

I created a Discord server for discussing anything related to the images. Here is your invite.

Gallery

The images below were written by hj2png. The numbers in the captions were the arguments of the options -i (ID) and -j (J-value), respectively. The option -z was given for additional compression.

The images are 512 pixels in width and height. The last two are grayscale, whereas the others are white-on-black.

There is an image from every class and main operation except ADD. Also, there is an equal number of AND and OR images.
