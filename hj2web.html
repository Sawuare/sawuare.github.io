hj2web

Description

hj2web is an interactive demo of HJ images. From the options below you can select a painter and input its parameters to paint a canvas. Note that the options are limited in comparison with those of the main program hj2png.

Favicon

This site's favicon is an HJ image that can be painted using a certain combination of the options below. Can you find it?

The combination

ID is 22, J-value is 15, and width and height are 16.

Painter 22 reacts to some J-value sequences, and J-value 15 begins one of them: 15, 62, 252, 1016, 4080, 16352…. Related sequences include A171472, A171499, and A171476. The program hjjs generates these sequences and more.
