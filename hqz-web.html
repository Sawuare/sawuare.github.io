hqz-web

hqz-web is an interactive demo of HQZ images. From the options below you can select a painter and input its parameters to paint a canvas. Note that the options are limited in comparison with those of the main program hqz-png.

Favicon

This site's favicon is an HQZ image that can be painted using a certain combination of the options below. Can you find it?

The combination

Painter is 22, q-argument is 15, and width and height are 16.

Painter 22 reacts to some q-sequences, and q-argument 15 begins one of them: 15, 62, 252, 1016, 4080, 16352…. Try these values below. Do you notice a pattern? Related sequences include A171472, A171499, and A171476. My program hqz-seq generates these sequences and more.
